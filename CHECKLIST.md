# CHECKLIST.md — 手動確認チェックリスト

Studio Play での動作確認用。実装・バグ修正のたびに影響する項目を更新する。

---

## Phase 1 — ゲームコア

- [ ] ゲーム開始時に53枚が4人に配られる
- [ ] ♦3保持者からスタートする
- [ ] ♦3スタート通知が表示される
- [ ] カードを選択→「出す」で場に出せる
- [ ] 場より弱いカード・枚数違いは出せない（サーバーが拒否）
- [ ] パスができる
- [ ] 全員パスで場がリセットされ「場が流れました」通知が出る
- [ ] NPCが自動でカードを出す/パスする
- [ ] 手札がなくなったら上がり、上がり通知が表示される
- [ ] 最後の1人が残った時点でラウンド終了

## Phase 2 — 特殊ルール

- [ ] 同じ数字4枚出しで革命が発生し「革命！」通知が出る
- [ ] 革命中はカードの強弱が逆転する
- [ ] 革命中に4枚出しで逆革命（元に戻る）
- [ ] 8を出すと「8切り！」通知、場にカードが表示された後リセットされる
- [ ] 8切り後、出したプレイヤーが続けて出せる
- [ ] ジョーカー1枚出しに♠3で返せる
- [ ] 反則あがり条件で上がると最下位になる（最強カード/8切り/スペ3返し/ジョーカー）
- [ ] NPCが反則あがりを回避する

## Phase 3 — セット・ポイント制・階級交換

- [ ] ラウンド番号が画面左上に「R1/4」形式で表示される
- [ ] 各プレイヤーのスコアが表示され、手番ごとに増加しない
- [ ] ラウンド終了時に結果オーバーレイが表示される
- [ ] 結果オーバーレイにタイマーバーが表示され減っていく
- [ ] 結果表示後、自動で次ラウンドに遷移する
- [ ] 2ラウンド目以降、階級交換が行われる
- [ ] 上位プレイヤー（大富豪/富豪）がカードを選んで交換できる
- [ ] 下位プレイヤーの最強カードが自動で上位に渡る
- [ ] 交換後、♦3保持者からスタートする
- [ ] 4ラウンド完了後に最終結果が表示される（タイマーバーなし）
- [ ] 最終結果のスコア順位が正しい
- [ ] ポイント配分: 1-3R（50/30/10/0）、4R（70/30/10/0）
